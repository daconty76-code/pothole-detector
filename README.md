# Pothole Detection System

Real-time pothole detection using YOLOv8n trained on RDD2022 dataset.

## Features
- Upload images for pothole detection
- Adjustable confidence threshold
- Real-time inference display

## Model Performance
- mAP@0.5: 0.317
- Inference speed: 48 FPS on GPU

## Classes Detected
- Longitudinal crack
- Transverse crack
- Alligator crack
- Other corruption
- Pothole
